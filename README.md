**Commander: Joint Assault** is a single-file, browser-based tactical simulation game featuring a retro CRT terminal aesthetic. Players command tanks or fighter jets in a real-time battle for territory, utilizing a mix of arcade shooting mechanics and real-time strategy elements.

This project is a unique experiment in software development: **The code, logic, and assets were entirely generated by Artificial Intelligence.**

### ğŸ¤– AI Attribution
This project was developed 100% through prompts and AI collaboration. No manual coding was performed by a human developer.
* **Code Generation:** OpenAI Codex
* **Logic & Refinement:** Google AI Studio (Gemini)

### âœ¨ Features
* **Single HTML File:** No external assets, images, or libraries required. Everything is procedurally generated via Canvas API and Web Audio API.
* **Joint Operations:** Seamlessly switch between ground units (Tanks) and air support (Jets).
* **Tactical AI:** Advanced AI agents that use A* pathfinding, flank enemies, retreat when damaged, and coordinate attacks.
* **Dynamic Audio:** Synthesized sound effects and dynamic background music generated in real-time.
* **Retro Aesthetics:** CRT scanline effects, pixel-art style rendering, and terminal interfaces.
